# booksync
Site do BookSync, ferramenta para organizar empréstimos e devoluções da biblioteca da ETEC.

Nesse site é possivel:
- Criar registros contendo informações locais como nome, sala, livro e data;
- Apagar registros com uma senha;
- Visualizar todos os registros já feitos nesse dispositivo;
- Salvar a lista de registros como um arquivo JSON;
- Filtrar lista de registros para certos requisitos.
